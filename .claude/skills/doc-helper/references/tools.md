# Available Tools Reference

## Read
Read file contents from the filesystem. Use to inspect source code, config files, or existing docs.

## Grep
Search file contents using regex patterns. Use to find references, usage examples, or verify terminology across the codebase.

## Glob
Find files by name pattern (e.g., `**/*.md`, `src/**/*.ts`). Use to discover existing documentation or related files.

## Bash
Execute shell commands. Use for:
- `git log` — check recent changes to a file
- `git blame` — see who last edited a section
- Running code examples to verify they work
- Listing directory structures for documentation
